<h2>CRUD em PHP</h2> 
<h3>Este é um projeto simples de CRUD (Create, Read, Update, Delete) desenvolvido em PHP com MySQL.
O objetivo é demonstrar operações básicas de banco de dados em uma aplicação web.</h3>

---

<h2>Tecnologias utilizadas:</h2>
* PHP
* MySQL
* HTML
* CSS

---

<h2>Funcionalidades:</h2>
* Criar novos registros (Create)
* Listar registros (Read)
* Atualizar registros (Update)
* Deletar registros (Delete)

---

<h2>Como executar o projeto</h2>

1. Clonar o repositório
* git clone https://github.com/ArthurQuintino/CRUD-PHP.git
2. Importar o banco de dados
* Abra o MySQL
* Crie uma database
3. Configurar conexão
* Edite o arquivo conexao.php com suas credenciais:

``` PHP
$server = "localhost";
$user = "root";
$password = "";
$bd = "sua database";
```
---

4. Rodar o projeto
* Coloque a pasta no htdocs (XAMPP) ou equivalente
* Acesse no navegador: http://localhost/CRUD-PHP
